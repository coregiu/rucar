# RUCAR
    利用51单片机、L298N驱动、PS2手柄和DIV小车实现的摇控音乐闪灯小车。

## 功能说明
#### 遥控小车行驶
可通过PS2控制小车前进、后退、左转、右转。

#### 受控音乐播放及切换
可通过PS2控制轩载音乐播放，音乐更换。

#### 遥控LED灯光效果切换
可通过PS2控制小车顶LED灯光变换。
灯光效果有常亮、跑马灯、间隔闪烁、互切、关闭等效果。
